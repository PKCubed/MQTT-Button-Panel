# Printing Instructions

## Buttons

* ### Transparent Part

Set infill to solid (100% density) and choose a pattern you like. Concentric infill gives a nice transparency. Choosing the largest layer height seems to give the best transparency. Below are the settings I used:
| Setting | Value |
| :--- | :--- |
| Layer height | 0.28mm |
| Wall loops | 1 |
| Top surface pattern | Concentric |
| Bottom surface pattern | Concentric |
| Internal solid infill pattern | Concentric |
| Infill pattern | Concentric |
| Infill density | 100% |
